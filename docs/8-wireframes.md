## 공통 레이아웃 원칙

### 브레이크포인트

| 구분 | 기준 | 레이아웃 |
|------|------|----------|
| 데스크탑 | >= 1024px | 사이드바(240px) + 메인 영역 2단 |
| 모바일 | < 1024px | 단일 컬럼, 하단 네비게이션 바 |

### 상태 배지 범례

| 배지 | 설명 | 조건 |
|------|------|------|
| `[NOT_STARTED]` | 시작 전 | is_done=false, 날짜 미설정 또는 today < start_date |
| `[IN_PROGRESS]` | 진행 중 | is_done=false, start_date <= today <= end_date |
| `[DONE]` | 완료 | is_done=true |
| `[OVERDUE]` | 기간 초과 | is_done=false, today > end_date |

### 공통 UI 기호

| 기호 | 의미 |
|------|------|
| `[ placeholder ]` | 텍스트 입력 필드 |
| `[버튼명]` | 클릭 가능한 버튼 |
| `▼` | 드롭다운 셀렉터 |
| `☑` / `☐` | 체크박스 선택됨 / 해제됨 |
| `◉` / `○` | 라디오 버튼 선택됨 / 해제됨 |
| `◀` / `▶` | 페이지네이션 이전 / 다음 |
| `─ ─ ─` | 구분선 |

| 항목 | 현재 상태 표시 | 변경 시 동작 |

| 테마 (다크) | `○ 라이트 모드  ◉ 다크 모드` | `userApi.updateSettings({ theme: "light" })` + `data-theme="light"` 즉시 적용 |
| 언어 (한국어) | `◉ 한국어 (ko)  ○ English (en)` | `userApi.updateSettings({ locale: "en" })` + `i18n.changeLanguage("en")` 즉시 적용 |
| 언어 (영어) | `○ 한국어 (ko)  ◉ English (en)` | `userApi.updateSettings({ locale: "ko" })` + `i18n.changeLanguage("ko")` 즉시 적용 |

### 주요 인터랙션 및 상태 변화

| 이벤트 | 동작 |
|--------|------|
| 테마 라디오 변경 | API 호출 → `authStore.user.theme` 업데이트 → `data-theme` 속성 즉시 변경 |
| 언어 라디오 변경 | API 호출 → `authStore.user.locale` 업데이트 → `i18n.changeLanguage()` 즉시 호출 |
| 설정 저장 실패 (네트워크 오류) | "설정을 저장하지 못했습니다." 에러 메시지 + 이전 상태로 롤백 |
| 이름 저장 버튼 | `authApi.updateMe({ name })` → 성공: `authStore.user.name` 갱신 |
| 비밀번호 변경 | `authApi.updateMe({ currentPassword, newPassword })` → 성공: 완료 메시지 |
| 현재 비밀번호 불일치 | HTTP 401 → "현재 비밀번호가 올바르지 않습니다." |
| 회원 탈퇴 버튼 | Modal 표시 (비밀번호 재확인) |
| 탈퇴 확인 | `authApi.deleteMe({ password })` → 성공: `authStore.clearAuth()` → `/login` 이동 |
